Homework 2 Rubric

These are the specific items homework one will be graded on. Not all of them are worth equal value. Use this a checklist to help you guide your testing and design.

Computer picks a random word of the length specified by the player.

Computer asks the player to input another length when it does not know any of the words that are of the length chosen by the player.

The number of lives starts at length of secret word + 4.

Asks player to input a letter guess.

Display a message when player enters a letter that has been guessed before (should be treated as an incorrect guess).

Incorrect guess deducts a life.

Correct guess updates the progress that the player has made towards guessing the word.

Game ends when player guesses all of the letters of the word correctly.

Game ends when player lives = 0.

(Extra Credit: Graphics) Add the ability to print the letters on the screen using sprites.  Account for word lengths up to and including 11 letters.

(Extra Credit: Algorithm) Instead of just stating the secret word, also include the number of occurrences within the word.

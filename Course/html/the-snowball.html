The Snowball

Now for the big part of the algorithm! Write a block combine strings, that takes in a list of strings and returns a single string that combines the list in a specific way:

  1. First, your block will search through the list to identify the two strings with the most overlap.
  2. Throw out the two strings you found and combine these two strings into their shortest superstring.
  3. Now, the process should be repeated with the superstring and the list of original strings that were not selected in the first step. This list will have one less string than in the first step.
  4. When there is only one string left, it should be reported.

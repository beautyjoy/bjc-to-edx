The Max Block

We will now make a different kind of block -- a reporter. To demonstrate this, you will make a block called max that takes two numbers as input and reports the bigger value (the maximum).



  1. Click Make a block and select the Operators palette. We want a reporter block. This will give the block its round shape as shown above. As the name implies, reporter blocks can report a value. In the image below, you can see that we used names starting with the % character as a shortcut for making input variables. Note: Usually, we want you to use meaningful variable names, so that your code will be self-documenting. But in this case, the inputs are two arbitrary numbers that don't have any special meaning inside the max block, so it's okay to use x and y.



  2. This should give you a blank Block editor. Notice that this time the script below the hat block already has a block in it, the report block from the Control palette. In order for your reporter block to report a value, you put that value (or an expression that produces the value) in the input slot of a report block. Note also that there is no tab at the bottom of the report block; when this block is run, it ends the script. There can be more than one report block in a script, as you'll see shortly, but only one of them can be run each time through the script.
